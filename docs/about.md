# About Kotlin FHIR Engine

## What is it?

A **Kotlin Multiplatform library** for building offline-capable healthcare applications using the HL7 FHIR R4 standard.

## Core Features

- **FHIR Resource Persistence** - Store FHIR resources locally
- **Remote Synchronization** - Sync with remote FHIR servers
- **Type-Safe Search API** - Build queries with compile-time safety
- **Offline-First** - Work without internet connection

## Supported Platforms

- ✅ **Android** - Full support
- ✅ **iOS** - Full support  
- ✅ **Desktop (JVM)** - Full support

## Part of OHS Player

This engine is a core component of the **Open Health Stack (OHS) Player** ecosystem.
